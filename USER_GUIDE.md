# Stock Predict — 用戶指南

## 歡迎使用 Stock Predict

Stock Predict 是一個港股/美股分析工具，結合**技術指標分析**和 **AI 智能分析**，幫助你做出更明智的投資決策。

---

## 快速開始

1. 打開瀏覽器訪問應用地址
2. 在左上方選擇或搜索一個股票代碼（例如 `0700.HK`）
3. 選擇分析週期（短線/中線/長線）
4. 點擊「技術分析」按鈕
5. 等待分析完成，查看結果

---

## 界面說明

### 頂部控制區

```
┌─────────────────────────────────────────────────┐
│ [股票選擇器]  [分析週期 ▼]  [技術分析]           │
└─────────────────────────────────────────────────┘
```

- **股票選擇器**：點擊下拉，可輸入股票代碼或名稱搜索，選擇後自動填入
- **分析週期**：控制技術分析和 LLM 分析的時間範圍
  - 短線 (1-5日)：適合日內或超短線交易
  - 中線 (1-4週)：適合波段操作
  - 長線 (1-3月)：適合中長期投資
- **技術分析**：點擊後同時執行技術指標分析和 AI 分析

### 自選股區域

```
┌─────────────────────────────────────────────────┐
│ 自選股                          [搜尋股票...]     │
├──────┬──────────────────────┬──────┬──────┬───┤
│ 代碼  │ 名稱                  │ 價格   │ 變動   │   │
├──────┼──────────────────────┼──────┼──────┼───┤
│0700.HK│ Tencent Holdings     │ 479.40│+3.54% │ ✕ │
│2899.HK│ Zijin Mining         │ 37.80 │+1.34% │ ✕ │
└──────┴──────────────────────┴──────┴──────┴───┘
```

- **搜索**：輸入股票代碼或公司名稱，自動搜索匹配的股票
- **添加**：搜索後點擊結果即可加入自選股
- **刪除**：點擊每行右側的 ✕ 按鈕移除
- **點擊行**：快速切換到該股票

## 圖表操作指南

### 切換時間框架

點擊圖表上方的時間框架按鈕：

| 按鈕 | 說明 | K線類型 | 數據範圍 |
|------|------|---------|---------|
| 分時 | 當日分時走勢 | 折線圖 | 最近一個交易日 |
| 5日 | 5日走勢 | 折線圖 | 最近5天 |
| 日線 | 日K線 | K線圖 | 最近2年 |
| 周線 | 周K線 | K線圖 | 最近5年 |
| 月線 | 月K線 | K線圖 | 最近10年 |
| 季K | 季K線 | K線圖 | 最近10年 |
| 年K | 年K線 | K線圖 | 最近20年 |

> **注意**：分時和5日模式下不支持主圖疊加指標（MA/EMA等），因為小時數據不適合這些指標。

### 主圖疊加指標

在日線及以上時間框架，可以疊加以下指標到K線圖上：

- **MA**：移動平均線（5/10/20/60日）
- **EMA**：指數移動平均線（5/10/20/60日）
- **BOLL**：布林帶（上軌/中軌/下軌）
- **SAR**：拋物線轉向指標（顯示為點狀）
- **KC**：肯特納通道（上軌/中軌/下軌）
- **一目**：一目均衡表（轉換線/基準線/先行帶A/先行帶B/遲行線）
- **VWAP**：成交量加權平均價

點擊按鈕開啟/關閉，可同時開啟多個。

### 副圖指標

所有時間框架都支持副圖指標，每個副圖在獨立的卡片中顯示：

| 指標 | 說明 |
|------|------|
| 成交量 | 默認開啟，紅綠柱狀圖 |
| MACD | MACD線/信號線/柱狀圖 |
| KDJ | K線/D線/J線 |
| ARBR | 人氣指標/意願指標 |
| CR | 中間意願指標 |
| DMA | 趨向移動平均線/AMA |
| EMV | 簡易波動指標 |
| RSI | 相對強弱指數（含70/30參考線） |

### 查看詳細數據

**鼠標懸停**在圖表上即可看到十字線工具提示：

- **主圖**：顯示日期、開盤/最高/最低/收盤價、成交量、漲跌幅
- **副圖**：顯示日期和對應指標的數值（如 MACD 的 MACD/信號線/柱狀圖值）

工具提示會自動避開邊緣，不會被遮擋。

## 分析結果

### 技術分析結果

點擊「技術分析」後，會顯示：

- **方向**：看多 🟢 / 看空 🔴 / 中性 ⚪
- **信心度**：0-100%
- **關鍵支撐/阻力位**
- **各指標的詳細評分和描述**

### AI 分析結果

LLM 分析會提供：

- **方向判斷**：看多/看空/中性
- **信心度**：0-100%
- **目標價格區間**：低目標 ~ 高目標
- **趨勢分析、動量分析、成交量分析**
- **看多/看空情景**
- **風險因素**

### 對比分析

技術分析和 LLM 分析都完成後，自動顯示對比視圖：

- **共識判斷**：兩者是否一致
- **綜合信心度**：加權平均
- **各方觀點摘要**

## 分析週期說明

分析週期控制的是**技術分析和 AI 分析的數據範圍**，與圖表的時間框架是獨立的：

| 週期 | 數據範圍 | 適用場景 |
|------|---------|---------|
| 短線 (1-5日) | 最近5天 | 日內交易、超短線 |
| 中線 (1-4週) | 最近20天 | 波段操作 |
| 長線 (1-3月) | 最近60天 | 中長期投資 |

> **提示**：圖表時間框架（分時/日線等）控制的是圖表顯示的數據範圍和K線類型，而分析週期控制的是分析信號使用的數據範圍。兩者獨立，可以自由組合。

## 常見問題

### Q: 為什麼分析很慢？

AI 分析需要調用 DeepSeek API，可能需要 30秒到5分鐘。請耐心等待。

### Q: 為什麼有些指標按鈕是灰色的？

分時和5日模式下，主圖疊加指標（MA/EMA/BOLL等）不可用，因為小時級別數據不適合這些日線指標。

### Q: 圖表沒有數據？

1. 確認後端服務已啟動（`uvicorn app.main:app --reload`）
2. 確認 `.env` 中的 `DEEPSEEK_API_KEY` 已正確設置
3. 檢查網絡連接，yfinance 需要訪問 Yahoo Finance API

### Q: 圖表上的日期格式為什麼有時不一致？

後端返回的日期格式可能混合 ISO 時間戳和 `YYYY-MM-DD` 格式。前端會自動處理這種不一致，將所有時間歸一化到統一格式。

### Q: 自選股列表存在哪裡？

自選股保存在後端的 `watchlist.json` 文件中，重啟後端不會丟失。

### Q: 支持哪些市場？

支持 yfinance 能獲取的所有股票，主要包括：
- 港股：代碼格式如 `0700.HK`、`9988.HK`
- 美股：代碼格式如 `AAPL`、`TSLA`
